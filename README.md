Photo Gallery Web App

A responsive photo gallery built as part of the Celebrare Frontend Intern Pre-Screening Assignment.
The app fetches images from a public API, supports real-time search filtering, and includes a modern responsive UI built with React and Tailwind CSS.

Live Features
Fetches 30+ photos from a public API
Real-time search by author name
Fully responsive grid layout (mobile, tablet, desktop)
Smooth UI interactions with hover effects
Clean component-based architecture using React hooks
Loading and error handling states

Tech Stack
React (Vite)
JavaScript (ES6+)
Tailwind CSS
Picsum Photos API

Project Setup

1. Clone the repository
   git clone https://github.com/your-username/gallery-app.git

2. Install dependencies
   npm install

3. Run the development server
   npm run dev

Key Concepts Implemented
React functional components
useState and useEffect hooks
Custom hook (useFetchPhotos)
Array filtering for search functionality
Responsive design using Tailwind CSS
Component re-rendering based on state changes
